- Tudo do Bloco 3: vigas, pilares, lajes, sapatas, impulsos e muros.

## Como usar

1. Usar preferencialmente num repositório novo/limpo.
2. Extrair o ZIP.
3. Enviar os ficheiros para o GitHub.
4. Executar o workflow **Build WebApp**.
