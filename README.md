- Lajes EC2 preliminar.

## Como usar no GitHub
1. Criar repositório novo ou limpar o antigo.
2. Extrair este ZIP.
3. Fazer upload de todos os ficheiros para a raiz do repositório.
4. Correr **Build WebApp**.
5. Só depois avançar para o Bloco 3.
